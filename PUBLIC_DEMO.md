# Public Demo Notice

This repository is intentionally a public portfolio demo, not a full production system.

Included:

- project structure
- basic PDF extraction flow
- OCR sample integration
- CSV export
- demo sample file

Excluded from public scope:

- advanced production pipelines
- client-specific business logic
- optimized extraction rules
- proprietary datasets

For private or production implementations, the core extraction logic is developed in a separate private workspace.
